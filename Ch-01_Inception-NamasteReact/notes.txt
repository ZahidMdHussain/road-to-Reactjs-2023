6. What is the difference between async and defer?
A. In async html parsing and script fetching happen in parallel, once the script is fetched then html parsing is 
   put on hold and execution of script starts, once the script is executed completely then again the html parsing 
   begins. And also in async the order of script execution is not considered.

   While in defer the html parsing and script fetching begin's in parallel same as async but once the script is
   fetched completely, our html parsing still continues and when the html parsing is completed then only the 
   script execution begins. And here the order of script execution is considered.
